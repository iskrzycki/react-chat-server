# react-chat-server

A node.js based, websocket server needed to launch react-chat application.
Project is based on [TicTacToe](https://github.com/iskrzycki/TicTacToe) web application made by [iskrzycki](https://github.com/iskrzycki)

### Installing

```
npm install
node app.js
```
